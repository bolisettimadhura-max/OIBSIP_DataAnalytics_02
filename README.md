Segment customers based on Annual Income and Spending Score using K-Means Clustering.

- NumPy
- Matplotlib

- Scikit-learn
- Google Colab

2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Scaling
5. K-Means Clustering
6. Elbow Method
7. Cluster Visualization

## Results
- Identified 5 customer segments
- Visualized customer groups successfully
